## Description
This simple application displays a list of Rick and Morty episodes. If the user taps on an episode another list appears showing the characters in that episode split into dead or alive and displayed in the order in which they were created. If the user taps on a character then a screen showing the characterâ€™s picture and information pops up.

Finally there is an additional functionality that allows the user to kill characters and everything in the model is updated accordingly.
